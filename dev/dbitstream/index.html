Deterministic Bitstreams

A DBitstream is a sequence of DBits representing PDM encoded audio.

BitSAD.DBit — Type
DBit

A deterministic bit is a single bit representing ±1.

Fields:

  • bit::Bool: a sample of a bitstream
source
BitSAD.DBitstream — Type
DBitstream

A deterministic bitstream that looks like a PDM-encoded audio format.

Fields:

  • bits::Queue{DBit}: the underlying bitstream
source

A DBit maps $\{0, 1\} \mapsto \{-1, 1\}$. We can access this mapping through float.

Base.float — Method
float(b::DBit)

Map b to the underlying floating-point value using $\{0, 1\} \to \{-1, 1\}$

source
